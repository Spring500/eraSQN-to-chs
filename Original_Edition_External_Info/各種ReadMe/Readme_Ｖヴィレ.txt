【利用規約】
SQNおよびSQ系統のバリアントに限り、再配布および転載を許可します。
また本口上の一部を他キャラへ転用・流用する事も許可します。
構文でもテキストでも好きに利用してください。

SQNおよびSQ系統以外のバリアントにおける利用、配布、転載、口上の転用・流用は一切禁止とさせて頂きます。

本口上への加筆は原則として自由ですが、作者の加筆の際に編集・削除の可能性があることはあらかじめご了承ください。
