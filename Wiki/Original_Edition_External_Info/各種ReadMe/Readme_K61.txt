僕の考えたヴァルキリー口上
原作のヴァルキリーについては良く知らないけど
自分のイメージで書いてしまった口上です。

真面目なイメージ、まじめだけど夢魔になっているのでエロいことには抵抗はない
相手を試すように振る舞うが、自分が認めた相手には忠誠を尽くす、騎士みたいな感じ
恋慕が付くと自分の欲求が出てきて少し恥ずかしがってみたり、甘えてみたりするようになる
普段から真面目なイメージで振る舞っているので、甘えたりしているところは見られたくない
屈服刻印Lv3からマスターと呼んでくれるようになる
また、要望などがあったら加筆したりするかもしれません
